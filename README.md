# Power Automate Reliability Kit

A reusable toolkit for building production-ready Power Automate flows.

The kit standardizes:
- TRY/CATCH/FINALLY lifecycle structure
- structured error logging
- run metadata capture (Flow Name, Run URL, timestamp)
- retry-safe execution patterns
- centralized failure visibility via Dataverse

## Kit Components
- Lifecycle Template (TRY / CATCH / FINALLY)
- Logging Sink (Dataverse example)
- Error Contract (standard schema)
- Testing Scenarios (reproducible failures)
